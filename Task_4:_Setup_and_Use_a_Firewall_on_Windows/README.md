# Setup and Use a Firewall on Windows

## About this Task
This task demonstrates how to configure and test basic firewall rules on a Windows system. The main objective is to block inbound traffic on a specific port (Telnet, port 23) and verify the rule works, then restore the original state by deleting the rule. This helps in understanding firewall management and how firewalls filter network traffic.

## Repo Structure
- **Report.md**: The main file containing a detailed walkthrough of all steps performed during the task, along with explanations and screenshots.  
- **Screenshots/**: This folder contains all the screenshots taken during the task.
