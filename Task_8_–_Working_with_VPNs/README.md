# Working with VPNs

This repository contains all files and documentation related to the **Working with VPNs** task, where I explored the use of VPNs to enhance **online privacy** and **secure communication**.  
The task involved installing a VPN client, connecting to a server, verifying IP address change, checking internet speed, and confirming secure browsing with screenshots.

## Repository Structure

- **Report.md** – The main file containing a detailed walkthrough of all steps performed during the task. This includes:  
  - Installing and signing in to ProtonVPN  
  - Checking IP and internet speed before connecting to VPN  
  - Connecting to a VPN server and verifying IP change  
  - Comparing internet speed before and after VPN  
  - Browsing websites to confirm secure traffic  
  - Research on VPN encryption, privacy features, benefits, and limitations  
  - Including screenshots for clarity  

- **Screenshots/** – Folder containing all screenshots captured during the task. These screenshots illustrate:  
  - VPN installation and login  
  - IP address before and after VPN connection  
  - Internet speed tests before and after VPN  
  - Websites browsed while connected to VPN  

- **README.md** – This file, providing a high-level overview of the task and the repository contents.

> **Note:** Only non-sensitive screenshots are included. No personal or confidential data is shared.
