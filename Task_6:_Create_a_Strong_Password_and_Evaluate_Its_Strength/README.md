# Create a Strong Password and Evaluate Its Strength

This repository contains all files and documentation related to the password strength testing task, where I evaluated various passwords using an **online password strength checker** and a **custom-built tool**.

## Repository Structure

- **Report.md** – The main file containing a detailed walkthrough of all steps performed during the task. This includes:  
  - Selecting and testing multiple passwords  
  - Using the online password strength checker [Password Monster](https://www.passwordmonster.com/)
  - Using the custom-built password strength tool from [PRODIGY_CS_03](https://github.com/cybersecurity-Pro/PRODIGY_CS_03) repository 
  - Comparing results and analyzing strengths  
  - Including screenshots for clarity  

- **Screenshots/** – Folder containing all screenshots captured during the task. These screenshots illustrate:  
  - Passwords being tested  
  - Results from the online tool  
  - Results from the custom-built tool  

- **README.md** – This file, providing a high-level overview of the task and the repository contents.  

> **Note:** Only non-sensitive screenshots are included. Passwords used are for testing purposes only, and no real personal credentials are shared.
